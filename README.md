🛡️ SmartSOS — Emergency Safety Platform

A full-stack women safety and emergency response web app 
built with the MERN stack (MongoDB, Express, React, Node.js).

✨ Features:
🚨 One-tap SOS button — sends WhatsApp emergency alert with 
   live GPS location, name and timestamp to all contacts
🤳 Shake to SOS — shake phone 3 times to trigger SOS 
   without touching screen
⏰ Safety Timer — auto-sends SOS if user doesn't check in
📍 Live Location Sharing — generates a real-time tracking 
   link sent to contacts via WhatsApp
🗺️ Live Map — find nearest hospitals, police stations and 
   safe places using Overpass API
📱 Fake Call — escape dangerous situations discreetly
👥 Emergency Contacts — manage and alert multiple contacts
📋 Safety Guides — emergency instructions and tips
🔐 Authentication — secure login with JWT + WhatsApp OTP

🛠️ Tech Stack:
Frontend  — React.js, React Router, Leaflet Maps
Backend   — Node.js, Express.js
Database  — MongoDB Atlas
Auth      — JWT, WhatsApp OTP verification
Maps      — OpenStreetMap, Overpass API, Leaflet.js
Alerts    — WhatsApp wa.me integration

☁️ Deployed on Render (backend) + Vercel (frontend)
